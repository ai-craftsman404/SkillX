# Run X

Use this skill to drive the app in a real environment.

## Scope
- Launch the target app or service.
- Exercise the relevant workflow.
- Confirm the actual runtime result.

## Workflow
1. Start the app or service.
2. Navigate to the target flow.
3. Interact with the UI or runtime endpoint.
4. Observe the result directly.

## Output
- Pass/fail result.
- Observed behavior.
- Logs or screenshots when relevant.

## Boundaries
- Do not substitute static code reading for runtime observation.
