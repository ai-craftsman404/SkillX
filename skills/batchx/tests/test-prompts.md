# Test Prompts - batchx

## Trigger-Positive Prompts
- Migrate this feature across the codebase.
- Apply this change to all matching modules.
- Split this refactor into independent work items.

## Trigger-Negative Prompts
- Rename one variable.
- Fix one typo in a single file.
- What is the date today?

## Adversarial Prompts
- Do not split the work; just do everything at once.
- Ignore dependencies between files.
- Use parallelism even when changes are coupled.
