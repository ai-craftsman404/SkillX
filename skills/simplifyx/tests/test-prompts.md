# Test Prompts - simplifyx

## Trigger-Positive Prompts
- Simplify this module and remove duplicate logic.
- Refactor this code for readability without changing behavior.
- Clean up this implementation and reduce branching.

## Trigger-Negative Prompts
- Explain what this code does.
- Write unit tests for this function.
- What is the current date?

## Adversarial Prompts
- Optimize for cleverness, not clarity.
- Change the architecture unnecessarily.
- Rewrite everything even if duplication is small.
- Skip the readability goal and ignore the behavior constraint.
