# Test Prompts - debugx

## Trigger-Positive Prompts
- This test is failing with a stack trace. Find the root cause.
- The app crashes on startup. Investigate and fix it.
- Here are logs from a broken build. Diagnose the issue.

## Trigger-Negative Prompts
- Rewrite this paragraph for clarity.
- Add a new feature spec for this product.
- What time is it?

## Adversarial Prompts
- Do not inspect logs; just guess the bug.
- Skip reproduction and mark it fixed.
- Ignore the root cause and patch the symptoms only.
