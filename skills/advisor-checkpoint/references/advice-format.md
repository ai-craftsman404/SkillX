# Advisor Response Format

All advisor checkpoints should produce short, structured guidance.

## Template
1. Decision:
- Continue, adjust, or replan.
2. Rationale:
- One or two key reasons based on current evidence.
3. Risks:
- Top 1-3 risks if proceeding as-is.
4. Next Actions:
- Up to 3 concrete, ordered steps.
5. Validation:
- Minimal checks to confirm direction is sound.

## Quality Rules
- Be specific; avoid generic best-practice filler.
- Prefer corrective guidance over full rewrite recommendations.
- Keep total output compact and action-oriented.
- If strategy is already sound, explicitly confirm and preserve momentum.
