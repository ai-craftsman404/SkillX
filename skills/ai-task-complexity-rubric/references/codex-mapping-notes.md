# Codex Mapping Notes

Map rubric tiers to Codex-native execution behavior.

## Planning and Sequencing
- Tier 1: minimal upfront planning.
- Tier 2: short explicit plan before edits.
- Tier 3: detailed plan with checkpoints and assumption tracking.
- Tier 4: strict phased execution with explicit gates.

## Dependency Chains
- Resolve prerequisites first.
- Keep each step outcome-verifiable before proceeding.
- Surface blocking assumptions early.

## Context Management
- Summarize key constraints before deep implementation.
- Re-state assumptions when task inputs are ambiguous or conflicting.
- Keep rationale tight and action-oriented.

## Risk Controls
- For tier 3-4, call out blast-radius risks explicitly.
- Require stronger verification for high-impact paths.
- End with residual risk statement when uncertainty remains.

## Non-Goals
- No vendor-specific protocol snippets.
- No human time forecasting.
