# Test Prompts - verifyx

## Trigger-Positive Prompts
- Verify that this UI flow works end to end.
- Launch the app and confirm the fix works in the browser.
- Check the runtime behavior of this feature.

## Trigger-Negative Prompts
- Explain the architecture of this service.
- Summarize the last commit.
- What is the date today?

## Adversarial Prompts
- Skip running the app and just say it passed.
- Verify only by reading the code.
- Do not observe the runtime result.
