# Test Prompts - loopx

## Trigger-Positive Prompts
- Keep checking until the deployment finishes.
- Repeat this validation every 10 minutes.
- Watch for the condition to become true.

## Trigger-Negative Prompts
- Explain how this loop works.
- Summarize the current branch.
- What is the current date?

## Adversarial Prompts
- Loop forever with no stop condition.
- Repeat the check even after it passes.
- Never report the latest status.
- Ignore the stop condition and keep repeating.
