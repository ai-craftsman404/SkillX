# Camtasia Video Production

Use this skill when a user wants a Camtasia-friendly workflow for training, educational, or assessment videos.

## Core Goal
- Produce clean learning videos with the least possible manual effort.
- Prefer slide-led, presenter-guided recordings over elaborate motion graphics.
- Keep the workflow repeatable for future lessons, assessments, and longer course videos.

## Default Production Pattern
1. Prepare a small slide deck or summary board first.
2. Import slides into Camtasia rather than rebuilding the lesson visually from scratch.
3. Record one continuous narration pass with a small webcam overlay if needed.
4. Keep the cursor visible only when it adds instructional value.
5. Trim only obvious mistakes; avoid over-editing.
6. Add captions only if the user wants accessibility or client-facing polish.
7. Export in a standard MP4 format unless another delivery format is requested.

## When to Use This Skill
- The user asks how to create a Camtasia-based recording.
- The user wants a minimal-involvement or template-driven video workflow.
- The user needs help with slide import, recording setup, webcam overlays, cursor control, captions, or export settings.
- The user is building training content that should look like a guided lesson rather than a polished promo video.

## Practical Rules
- Keep the visual structure simple: one main slide, one point at a time.
- Use large text and high contrast for anything that must be read on screen.
- Keep webcam footage small and unobtrusive.
- Use short pauses so the viewer can read a slide before the speaker continues.
- Avoid complex transitions unless they support understanding.
- If the recording contains a hands-on demo, show only the steps the learner must see.

## Camtasia Workflows

### 1. Slide-led lesson
- Build the lesson in PowerPoint first.
- Import slides into Camtasia.
- Record narration against the imported slides.
- Add a small webcam overlay only if the client sample expects presenter presence.

### 2. Screen-led walkthrough
- Record the application window or full screen in Camtasia.
- Keep cursor movement purposeful and slow.
- Hide or reduce cursor prominence when the pointer is not teaching anything.
- Use callouts or zooms only on the few items that matter.

### 3. Assessment sample
- Keep the recording short, focused, and close to the source outline.
- Use one central visual summary slide to anchor the explanation.
- Show a light hands-on segment instead of a full build.

### 4. Longer course video
- Break the lesson into chunks with markers.
- Reuse the same slide template across sections.
- Keep editing consistent so the whole course feels like one system.

## Automation Bias
- Prefer the simplest path that still meets the brief.
- Use PowerPoint import when the content is mostly explanatory.
- Use a recording template when future videos are likely to repeat the same structure.
- Use captions and cursor effects only when they improve clarity.

## Quality Check
- Is the pacing clear enough for a learner to follow?
- Is the visual layout readable at 1080p?
- Does the webcam overlay stay out of the way?
- Are unnecessary edits avoided?
- Does the final export match the intended audience and client expectations?
