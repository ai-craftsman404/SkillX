# Camtasia Workflow Notes

## Best-fit workflow for this project
- Build a short PowerPoint deck first.
- Import the slides into Camtasia.
- Record one clean narration pass.
- Keep a small webcam overlay in the corner when needed.
- Use one central visual summary slide to explain the main concept.
- Minimise cursor noise and unnecessary motion.
- Export a simple MP4 with captions only if required.

## Advanced use cases
- Picture-in-picture presenter overlays.
- Cursor emphasis and click highlighting.
- Timeline markers for chapter-style navigation.
- Reusable templates for multi-lesson courses.
- Burned-in captions for client-facing delivery.
